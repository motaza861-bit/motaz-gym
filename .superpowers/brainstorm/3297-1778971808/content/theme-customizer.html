How it works technically
• One CSS variable --accent replaces all the hardcoded color values across every page
• Card style switches a data-card-style attribute on <html> — glass vs flat is handled by CSS
• Background sets --bg and --bg-card from 4 presets
• Settings page gets a new "Appearance" section with a native color picker + the toggles shown above
• Changes apply instantly — no reload needed
